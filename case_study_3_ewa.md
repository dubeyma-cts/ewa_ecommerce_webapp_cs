# 1.         Introduction

## 1.1       Overview

This case study is of an e-commerce website, [http://www.bidorbuy.com](http://www.bidorbuy.com/) that will allow the buyer to either bid for an item or directly buy it out without bidding. There will be 3 types of users: Buyers, Sellers & Administrators. Administrators are internal users & Buyers & Sellers are external.

The Sellers will be able to setup their shop on the site & sell their products. There will be different modes of payment available to the buyer. Facility of payment gateway is also provided to the seller.

The Administrators will be able to administer the site &manage Buyers & Sellers.

# 2.         Home Page

The Homepage of the site will let the user view all the Items that are on sale on the site. The items will be displayed according to their category. Users will be able to view the details of the item by clicking on the image; however, they will have to login to transact.

The home page will have following sections:

## 2.1       General Area

The user can view all the Items on sale by the Sellers in General Area. The General area will have Item images as well as Category listings. Images of Top 5 selling items will be displayed in this area, for rest of the items, the user will have to click on the Category listings, for e.g. if mobile phones, cameras, DVDs, laptops & toys are top selling items on bidorbuy, images of only those items will be displayed in the General Work Area. After clicking on an image, the user will be shown all the items of that category on sale on the site. If the user has to view an item which is not in the Top-5 list, say Garments, he/she will have to click on the Category in the Category listing to view the details. The user will be able to transact only after logging on to the site.

The site will allow the sellers to sell items of following categories:

·       Antiques: Antique items, artifacts & figurines

·       Art: Paintings & Sculptures

·       Baby Products: New born products

·       Books: Story books, Novels, Technology books

·       Cameras: Digital cameras, Video cameras

·       Mobile Phones

·       DVDs: Music DVDs, Movie DVDs

·       Toys: Plastic toys, stuffed toys, educational toys

·       Computers: Desktop computers, Notebook computers, Computer peripherals

·       Watches: Wristwatches, table clocks

·       Jewelry: Gold, Silver, Diamond, Artificial jewelry

·       Electronics: Televisions, Music Systems, DVD Players

·       Home Appliances: Refrigerators, Washing machines, Microwave ovens, OTG, Dish washers, Vacuum cleaners, Toasters, Water Purifiers, Iron.

## 2.2       Search Facility

The user will be able to use the search facility to search for an Item or a Seller. Based on the option selected, the user will be displayed a listing of either the Item typed in the search box or the Seller. Items can also be searched based on category. The user will also be provided with an ‘Advanced Search’ option using which he/she will be able to search for items based on their Category, Price, Condition (new/used), Seller Name, Seller location, Bid formats (Auction/Buy it Now)

## 2.3       Login Section

Buyers will be able to buy items on the site only after logging on to the site. Similarly, the Sellers will be able to add items to their shop only after logging on to the site.

# 3.         Buyers Area

Registered users who want to buy items from bidorbuy.com will be redirected to this section after logging on to the site.

Buyers Area will have following sections:

## 3.1       Buyers Home-Page

Users who want to buy or bid will be redirected to this page after logging on to the site. Just as in General Area, images of Top 5 selling items will be displayed on Buyers Home page. Rest of the items will be displayed according to their categories in Category listings. After clicking on an image, the user will be shown all the items of that category on sale on the site. If the user has to view an item which is not in the Top-5 list he/she will have to click on the Category in the Category listing. Buyers Home-page will also have links to following sections: View Items, Search Items, Bid for an Item, Buy an Item, Compare Prices & View Past Transactions.

## 3.2       View Items page

This page will display all the items of a particular category when the user either clicks on the image of the category or clicks on the category name from the Category listing on Buyers Home-page, for e.g. if the user clicks on Camera image, all the listings for Cameras will be displayed on View Items page.

The page will display a list containing following details:

·       Item Image

·       Item Name

·       Item Description

·       Latest Bid price

·       Bid Start Date

·       Bid End Date

·       Bid End Time

·       Buy price

·       Shipping Cost

·       Sellers Name

·       Sellers Location

The user can click on one item listing to either bid or straight away buy that item.

## 3.3       Search Items

Users will be able to search for an Item by typing the name of that item in the search box & clicking on the search button, for e.g. if a user wants to search for mobile phones, or for a particular brand of mobile phone, he/she can type the search criteria in the search box & click on the Search button. All the mobile phones matching the search criteria will be displayed on View Items Page. The user can then click on one item listing to either bid or straight away buy that item. The user will also be provided with an ‘Advanced Search’ option using which he/she will be able to search for items based on their Category, Price, Condition (new/used), Seller Name, Seller location, Bid formats (Auction/Buy it Now)

## 3.4       Bid for an Item

The user will be able to bid for an item by clicking on it from the item listings on View Items page. The user will be displayed Bid for an Item page that will show following details:

·       Image of the Item

·       Item Name

·       Description of the Item

·       Item Condition (New/Used)

·       Seller of the Item

·       Seller Location

·       Shipping Cost

·       Current Highest bid

·       Current Highest Bidder

·       Bid Start Date

·       Bid End Date

·       Bid End Time

·       Place your Bid

·       Confirm Bid (button)

·       Buy the Item (button)

·       Buy Price

Once the User places his/her bid, if it is the highest bid, the user’s name will be displayed in the Current Highest Bidder field. The highest bidder will be declared a Winner on completion of Bid time if there is no minimum threshold value set by the seller. If the Seller sets a minimum threshold value, the highest bidder will be declared a winner only if his/her bid value is higher than or equal to the minimum threshold value or else the bid will expire without a winner & the item will remain unsold. Payment can be done through multiple means as explained in section 7.1

## 3.5       My Current Bids page

The user will be able to see all his/her current bids on this page. Following details will be displayed on this page:

·       Item Image

·       Item Name

·       Item Description

·       Seller Name

·       Current Highest Bid

·       My Bid

·       Bid End Date

·       Bid End Time

·       Bid Status – (Active/Expired)

·       Renew my bid (button – If the bid is still Active)

## 3.6       Buy an Item

The user will also be able to buy an item directly instead of bidding for it by clicking on it from the item listings on View Items page & then clicking the ‘Buy the Item’ button present on the underlying Bid for an Item page that is displayed. The User will be shown following details on the Bid for an Item page:

·       Image of the Item

·       Item Name

·       Description of the Item

·       Item Condition (New/Used)

·       Seller of the Item

·       Seller Location

·       Shipping Cost

·       Current Highest bid

·       Current Highest Bidder

·       Bid Start Date

·       Bid End Date

·       Bid End Time

·       Place your Bid

·       Confirm Bid (button)

·       Buy the Item (button)

·       Buy Price

Once the user directly buys the item, all the bids for that item from different users, including the highest bidder, will be cancelled. Once the item is bought the user can make payment through multiple means as explained in section 7.1

## 3.7       Compare Prices

If the user decides to directly buy a product instead of bidding for it, he/she will be able to do a comparison of prices for that product from different sellers. In order to do a comparison the user will have to choose multiple items from the Items listings on View Items Page & click on ‘Compare’ button at the bottom of the listing. The user will be displayed following details on a new page:

·       Item Image

·       Item Name

·       Item Description

·       Seller Name

·       Seller Location

·       Buy Price

·       Shipping cost

After the comparison, the user can choose a seller & make payments through multiple means as explained in section 7.1

## 3.8       View Past Transactions

The users will be able to view all the past winning transactions as well as direct buys on this page. The page will display a list containing following details:

·       Item Image

·       Item Name

·       Item Description

·       Seller Name

·       Seller Location

·       Date of transaction

·       Item Cost

·       Bid/Buy

## 3.9       Watch an Item

Users will be able to watch the progress of bidding of a particular item by keeping it under a ‘Watch’. By doing this the user will be able to directly access the item under watch without wasting any time searching for it from the entire list of items displayed on View Items Page. The user will be able to put an item under watch by clicking on ‘Watch this Item’ button on the Bid for an Item page. The user will be able to watch all the items that he/she has placed under watch on the My Watch List Page.

## 3.10   My Watch List page

The User will be able to watch all the Items under his/her watch on this page. This page will display a list containing following details:

·       Item Image

·       Item Name

·       Item Description

·       Seller Name

·       Seller Location

·       Current Highest bid

·       Bid End Date

·       Bid End Time

The user will be able to place bid for a particular item by clicking on the Item image.

# 4.         Sellers Area

Registered Sellers who want to sell their items on bidorbuy.com will be redirected to this section after logging on to the site.

Sellers will be able to setup their shop only after their account is activated on bidorbuy.com. After registering, each seller will have to pay rent to bidorbuy.com to activate his/her account. The rent can be paid on monthly, quarterly or yearly basis. The monies can be paid through various different payment options. The administrator will activate the account only after the payment is received. The seller will be able to transact only after the shop is set on the site.

Following details will have to be entered by the seller to Setup a shop:

·       Shop Name

·       Shop Description

·       Acceptable Payment Options (Credit card, Cheque, Demand Draft, Cash On Delivery)

## 4.2       Add/Edit Items to Shop

The seller will be able to add items to shop only after the shop is setup on bidorbuy.com. Following details will be added:

·       Item Quantity

·       Item Description

Once the items are dispatched the quantity of each item will be automatically decremented. If the quantity reaches zero after dispatch, the item will be automatically removed from the list of the shop & the seller will be informed via an automatic mail. The seller will be able to update quantity of each item using this page. The seller will be shown following details:

·       Item Image

·       Item Name

·       Item Description

·       Item Price

·       Item Category

·       Item Quantity

·       Minimum Threshold Price

·       Shipping cost

·       Item Condition

The seller will also be able to edit other details of the items from this page.

## 4.7       View Past Transactions

The sellers will be able to view all the past transactions on this page. The page will display a list containing following details:

·       Item Image

·       Item Description

·       Buyer’s Name

·       Shipping Address

·       Item Selling Price

·       Date of transaction

·       Payment option chosen

·       Bid/Buy

## 4.8       Pay to bidorbuy.com

The seller will be able to transact only after the shop is set on the site.

# 7.         Additional Information

## 7.1       Payment Options

The buyers will be able to pay through various different options:

·       Paypal: Users can use their Paypal accounts to make payments for their purchases

·       Credit card: Users can use their credit-cards to make payments for their purchases. This will be done through a payment gateway

·       Demand Draft (DD): The buyer can send a demand draft for the purchase. The seller will dispatch items after the DD is deposited & encashed.

·       Cheque Payment: The buyer can send a bank cheque for the purchase. The seller will dispatch items after the cheque is deposited & encashed.

# 8.         Special Considerations

Following important requirements will also have to be considered:

·       The Application should be maintainable

·       The design should be scalable

·       Zero down time

·       24\*7 Availability

·       Should be able to handle heavy traffic load

·       Should interface with internal systems
