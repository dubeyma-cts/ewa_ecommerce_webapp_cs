| [UC11](#uc11) | Past Transactions | CSV/Excel export; show refunds; retain 24 months.  | [UC11](#uc11): Transaction history.  | Export file; refund entries; archive after 24m. | Confirmed |
| [UC12](#uc12) | Setup Shop | Logo optional; no vanity URLs; description ≤ 1,000 chars.  | [UC12](#uc12): Configure shop profile.  | Field constraints; autogenerated shop IDs. | Confirmed |
| [UC13](#uc13) | Item Listing | ≤ 6 images; JPG/PNG only; category auto-complete.  | [UC13](#uc13): Add/Edit item.  | Upload/type validations; auto-complete works. | Confirmed |
| [UC14](#uc14) | Auction | 1–14 days; reserve hidden; no auto-extend.  | [UC14](#uc14): Schedule auction & threshold.  | Enforce window; reserve concealed; hard end time. | Confirmed |
| [UC15](#uc15) | Sale Status | Limited buyer info before payment; show ETA if provided; show partial shipments.  | [UC15](#uc15): Status board & fulfillment details.  | PII masking; ETA field; multiple dispatch rows. | Confirmed |
| [UC16](#uc16) | Dispatch | Start with manual courier; no partial dispatch in MVP; barcode optional.  | [UC16](#uc16): Dispatch flow & notifications.  | Prevent split dispatch; capture carrier+tracking; stock auto-decrement. | Confirmed |
| [UC17](#uc17) | Stock Update | Low stock ≤5 default (configurable); bulk update via CSV/UI; auto-remove at zero.  | [UC17](#uc17): Update qty and zero→remove.  | Threshold alert; CSV import; zero-qty removal. | Confirmed |
| [UC18](#uc18) | Rent Payment | 7-day grace; PayPal/Cards/Cheque/DD; no auto-debit, send reminders.  | [UC18](#uc18): Pay to platform; admin activates after payment.  | Payment methods; grace window; reminder scheduler. | Confirmed |
| [UC19](#uc19) | Seller Activation | Notify seller on status change; single-admin approval sufficient.  | [UC19](#uc19): Activate/Deactivate seller.  | Notification events; admin audit logs. | Confirmed |
| [UC20](#uc20) | Transaction History | CSV/Excel export; include gateway reference IDs; multi-select filters.  | [UC20](#uc20): Admin transaction view.  | Export content; filter UX; gateway ID presence. | Confirmed |
| [UC21](#uc21) | Rent Defaulters | Weekly auto-reminders; 3-level escalation (reminder→warning→suspension candidate).  | [UC21](#uc21): Default rent payments.  | Scheduler cadence; escalation log per stage. | Confirmed |



# Functional — STRIDE Threat-to-Control Mapping

| STRIDE | Example Threat (Functional scope) | Impact on UC(s) | Control / Decision | How to Validate | Source |
|---|---|---|---|---|---|
| S (Spoofing) | Impersonate seller/admin to change status | [UC19](#uc19) | MFA for Seller/Admin; notify seller on status change; RBAC.  | Attempt role misuse; verify MFA and notifications. |  |
| T (Tampering) | Manipulate bid increment/client-side values | [UC6](#uc6) | Server enforces min increment = max(5%, ₹10); rejects under-min bids.  | Negative API tests. |  |
| R (Repudiation) | Buyer denies Buy-Now | [UC7](#uc7) | Gateway reference IDs + audit; atomic bid-cancel on buy.  | Check admin history for gateway IDs; audit trail. |  |
| I (Info Disclosure) | Reveal bidder PII | [UC6](#uc6), [UC15](#uc15) | Show bidder handle only; limit buyer info pre-payment.  | UI/API masking tests. |  |
| D (DoS) | Bid page refresh overload | [UC10](#uc10) | 5s auto-refresh cadence; avoid competitor counts to limit load.  | Load test [UC10](#uc10); observe QPS. |  |
| E (EoP) | Buyer triggers admin-only exports | [UC20](#uc20) | RBAC; admin-only export; include gateway IDs for audit.  | Role tests; export visibility. |  |

# Non-Functional — Requirements Traceability Matrix (RTM)

| Use-Case ID | NFR Area | Confirmed Decision (from Clarifications) | Linked NFR UC / BRD Context | Verification Approach | Status |
|---|---|---|---|---|---|
| [UC-NFR1](#uc-nfr1) | Availability | Zonal failover primary; regional failover secondary; health probes 5s(app)/15s(DB); zero-downtime via blue/green + smoke tests.  | [UC-NFR1](#uc-nfr1): HA DB + rolling/blue-green.  | Chaos tests; probe intervals; canary + smoke suite. | Confirmed |
| [UC-NFR2](#uc-nfr2) | Performance | p95 ≤3s UI & ≤500ms API; CDN cache for public GET; plan 10k concurrent sessions.  | [UC-NFR2](#uc-nfr2): CDN, cache, autoscale.  | Load/spike tests; cache headers; concurrency sim. | Confirmed |
| [UC-NFR3](#uc-nfr3) | Security | MFA mandatory (Admin/Seller); audit retention 12m; SSE-KMS at rest + TLS in transit.  | [UC-NFR3](#uc-nfr3): WAF, OIDC, KMS, audit.  | MFA policy; encryption config; retention jobs. | Confirmed |
| [UC-NFR4](#uc-nfr4) | Usability | WCAG AA across buyer/seller; theme contrast enforced; inline validation.  | [UC-NFR4](#uc-nfr4): A11y & responsive UI.  | Axe scan; manual audit; keyboard-only flows. | Confirmed |
| [UC-NFR5](#uc-nfr5) | Reliability | Friendly retry (no rollback details); exponential backoff 100ms–2s×3; daily reconciliation + hourly delta.  | [UC-NFR5](#uc-nfr5): ACID + idempotency + outbox.  | Fault injection; idempotency; reconciliation tasks. | Confirmed |
| [UC-NFR6](#uc-nfr6) | Maintainability | Contract tests blocking; rolling updates; versioned APIs (semver).  | [UC-NFR6](#uc-nfr6): Modular services + CI/CD.  | CI gate; canary rollout; version negotiation. | Confirmed |
| [UC-NFR7](#uc-nfr7) | Scalability | Scale-out at CPU≥70%/3m & Mem≥75%; queue depth >100 adds workers; safe scale-in at CPU<40% without spikes.  | [UC-NFR7](#uc-nfr7): Autoscaler + LB.  | Autoscaler policy tests; queue thresholds; guardrails. | Confirmed |
| [UC-NFR8](#uc-nfr8) | Observability | Alerts on p95 latency, error<1%, uptime ≥99.9%; 100% tracing for checkout/bidding/payment; runbooks required.  | [UC-NFR8](#uc-nfr8): Logs/metrics/traces & alerting.  | SLO dashboards; trace sampling; runbook links. | Confirmed |



# Non-Functional — STRIDE Threat-to-Control Mapping

| STRIDE | Example Threat (Platform/NFR scope) | Impacted NFR UC | Control / Decision | How to Validate | Source |
|---|---|---|---|---|---|
| S (Spoofing) | Session/token theft at edge | [UC-NFR3](#uc-nfr3) | MFA (Admin/Seller); WAF; OIDC token validation; TLS everywhere.  | Pen-test; MFA policy; TLS cert checks; WAF rules. |  |
| T (Tampering) | Data changed in transit/at rest | [UC-NFR3](#uc-nfr3), [UC-NFR5](#uc-nfr5) | TLS in transit; SSE-KMS at rest; ACID txns; outbox pattern.  | Inspect ciphers; KMS keys; partial-failure simulations. |  |
| R (Repudiation) | User denies action (bid/buy/admin) | [UC-NFR3](#uc-nfr3), [UC-NFR8](#uc-nfr8) | Immutable audit logs; 12m retention; correlation-ids in traces.  | Audit review; retention job; trace stitching. |  |
| I (Info Disclosure) | PII leakage via logs/exports | [UC-NFR3](#uc-nfr3), [UC-NFR8](#uc-nfr8) | PII-safe logging; role-based export; UI masking.  | Log sampling; RBAC tests on export endpoints. |  |
| D (DoS) | Traffic spikes & hot partitions | [UC-NFR1](#uc-nfr1), [UC-NFR2](#uc-nfr2), [UC-NFR7](#uc-nfr7) | Zonal/regional failover; autoscale thresholds; CDN caching; queue-depth scaling.  | Chaos/soak tests; autoscaler policy; CDN hit-ratio. |  |
| E (EoP) | Misconfigured roles or CI/CD | [UC-NFR3](#uc-nfr3), [UC-NFR6](#uc-nfr6) | RBAC by role; contract tests blocking; pipeline approvals/SOD.  | Role matrix tests; contract-breakers block merges. |  |
